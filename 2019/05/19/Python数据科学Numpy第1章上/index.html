Python整型不仅仅是一个整型

Python 3.4 中的一个整型实际上包括 4 个部分。

  • ob_refcnt 是一个引用计数，它帮助 Python 默默地处理内存的分配和回收。
  • ob_type 将变量的类型编码。
  • ob_size 指定接下来的数据成员的大小。
  • ob_digit 包含我们希望 Python 变量表示的实际整型值。
